# vibium_for_snow
Exploring Vibium library to automate UI action son ServiceNow application; and to check how it helps or where to add additional layers of code.

## Google AntiGravity
-  Used to generate the code for the UI actions

## Vibium
-  Used to automate the UI actions

## Prerequisites
- Python 3.8 or higher
- ServiceNow instance
- Vibium library
- Google AntiGravity

## Installation
- Clone the repository
- Install the required dependencies
